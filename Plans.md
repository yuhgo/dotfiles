## マーカー凡例

| マーカー | 意味 |
|---------|------|
| `pm:依頼中` | PM から依頼されたタスク（ソロPM Claude / Cursor PM 共通） |
| `cc:TODO` | Claude Code 未着手 |
| `cc:WIP` | Claude Code 作業中 |
| `cc:完了` | Claude Code 完了（確認待ち） |
| `pm:確認済` | PM 確認完了 |
| `cursor:依頼中` | （互換）`pm:依頼中` と同義 |
| `cursor:確認済` | （互換）`pm:確認済` と同義 |
| `blocked` | ブロック中（理由を併記） |

## 拡張記法（オプション）

大規模タスクの管理には、以下の記法を**オプションで**使用できます。

### タスク ID / 依存関係 / 並列可

```markdown
- [ ] T001: 認証機能 `cc:TODO`
- [ ] T002: ユーザーAPI `cc:TODO` depends:T001
- [ ] T003: 商品API `cc:TODO` [P]
- [ ] T004: 注文API `cc:TODO` depends:T001,T003
```

| 記法 | 意味 | 例 |
|------|------|-----|
| `T001:` | タスクID（任意） | 参照・依存指定に利用 |
| `depends:ID` | 依存タスク | `depends:T001,T002` |
| `[P]` | 並列可（Parallelizable） | 他タスクと同時実行可 |
