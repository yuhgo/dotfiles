# 売上・入金CSV一括作成スキル

取引先の月別売上金額データ（画像・テキスト・手書きメモなど）を元に、**売上CSV**と**入金CSV**の2ファイルを一括で作成する。

## 前提条件

- 出力先プロジェクト: 確定申告用のリポジトリ（`csv/` フォルダが存在すること）
- エンコーディング: **Shift_JIS**（会計ソフト互換）
- フォーマット: **全フィールドをダブルクォートで囲む**
- 金額のカンマ区切りは除去して数値のみにする（例: `660,094` → `660094`）

## CSVヘッダー（固定19列）

```
"取引No","取引日","借方勘定科目","借方補助科目","借方部門","借方取引先","借方税区分","借方インボイス","借方金額(円)","貸方勘定科目","貸方補助科目","貸方部門","貸方取引先","貸方税区分","貸方インボイス","貸方金額(円)","摘要","タグ","メモ"
```

## 入力データの形式

ユーザーは以下のいずれかの形式で月別売上金額を提供する:

- **画像（手書きメモ・写真）**: 月番号と金額の一覧
- **テキスト**: 月ごとの金額リスト
- **テーブル形式**: TSVやMarkdownテーブル

### 読み取るべき情報

1. **取引先名** — フォルダ名・ファイル名・摘要に使用
2. **月別金額** — 1月〜12月の売上金額（0円の月はスキップ）
3. **売上計上日** — 毎月の何日に計上するか（デフォルト: 20日）
4. **入金日** — 売上月の翌月何日に入金されるか（ユーザーに確認）
5. **入金の仕訳パターン** — 振込手数料の有無（ユーザーに確認）
6. **前年12月分の入金** — 年度をまたぐ入金があるか（ユーザーに確認）

## 実行フロー

### Step 1: データの読み取り・解析

画像やテキストから取引先名と月別金額を読み取る。読み取った内容をテーブルでユーザーに提示して確認する。

### Step 2: ユーザーへの確認

以下を確認する（ARGUMENTSで指定されている場合はスキップ可）:

#### 必須確認事項
- **取引先名**: フォルダ名・ファイル名に使用（例: `富田通商`）
- **年度**: 何年度の確定申告データか（例: `2025`）
- **売上計上日**: 毎月何日に売上を計上するか（デフォルト: 20日）
- **入金日ルール**: 売上月の翌月何日に入金されるか（例: `翌月25日`）
- **入金の仕訳パターン**: 振込手数料ありか、なしか

#### 状況に応じた確認事項
- **前年12月分の入金**: 当年度に含める前年売上の入金があるか（金額・入金日）
- **当年12月分の入金**: 翌年になるため当年度CSVには含めないことの確認
- **0円の月の扱い**: 売上が0円の月はCSVから除外することの確認

### Step 3: 売上CSVの生成

```
仕訳パターン:
  借方: 売掛金
  貸方: 売上高
  取引日: 毎月の売上計上日（例: 20日）
  摘要: <取引先名> <月>月分
```

**ルール:**
- 売上金額が0円の月はCSVに含めない
- 取引Noは1から連番

### Step 4: 入金CSVの生成

```
仕訳パターン（手数料なし）:
  借方: 普通預金
  貸方: 売掛金
  取引日: 翌月の入金日（例: 翌月25日）
  摘要: <取引先名> <月>月分入金

仕訳パターン（手数料あり）:
  行1:
    借方: 普通預金
    貸方: 売掛金
    金額: 売上金額 - 手数料
  行2:
    借方: 支払手数料
    貸方: 売掛金
    金額: 手数料
  摘要: <取引先名> <月>月分入金
```

**ルール:**
- 売上が0円の月は入金もスキップ
- 前年12月分の入金がある場合、先頭に追加（摘要に「前年度/12月分入金」等と記載）
- 当年12月分の入金は翌年になるため、当年度CSVには**含めない**
- 取引Noは1から連番

### Step 5: CSVファイルの生成（Python3）

```python
import csv
import os

base_dir = "<プロジェクトルート>/csv"
target_dir = os.path.join(base_dir, "<取引先名>")
os.makedirs(target_dir, exist_ok=True)

header = [
    "取引No","取引日","借方勘定科目","借方補助科目","借方部門",
    "借方取引先","借方税区分","借方インボイス","借方金額(円)",
    "貸方勘定科目","貸方補助科目","貸方部門","貸方取引先",
    "貸方税区分","貸方インボイス","貸方金額(円)","摘要","タグ","メモ"
]

# 売上CSV
sales_data = [
    # 0円の月を除いた売上データ
]

sales_path = os.path.join(target_dir, "売上_<取引先名>.csv")
with open(sales_path, "w", encoding="shift_jis", newline="") as f:
    writer = csv.writer(f, quoting=csv.QUOTE_ALL)
    writer.writerow(header)
    writer.writerows(sales_data)

# 入金CSV
payment_data = [
    # 前年12月分入金（あれば先頭に）
    # 当年1月〜11月売上の入金（12月分は翌年のため除外）
]

payment_path = os.path.join(target_dir, "入金_<取引先名>.csv")
with open(payment_path, "w", encoding="shift_jis", newline="") as f:
    writer = csv.writer(f, quoting=csv.QUOTE_ALL)
    writer.writerow(header)
    writer.writerows(payment_data)
```

### Step 6: 出力確認

生成した2つのCSVファイルを `iconv -f Shift_JIS -t UTF-8` で表示し、ユーザーに内容を確認してもらう。

## データ変換ルール

| 入力 | 出力 | 例 |
|------|------|-----|
| カンマ付き金額 | カンマ除去 | `660,094` → `660094` |
| 空セル | 空文字列 `""` | |
| タブ区切りの入力 | CSVカンマ区切りに変換 | |

## フォルダ構成・命名規則

```
csv/
└── <取引先名>/
    ├── 売上_<取引先名>.csv
    └── 入金_<取引先名>.csv
```

## エラーハンドリング

| ケース | 対応 |
|--------|------|
| フォルダが既に存在する | そのまま使用 |
| 同名ファイルが既に存在する | ユーザーに上書きするか確認 |
| 画像の文字が読み取りにくい | 読み取った内容をテーブルで提示し、ユーザーに確認 |
| 金額が数値でない | ユーザーに確認 |

## 具体例

### 入力例
画像またはテキストで以下のような月別データを受け取る:

```
富田通商
1   0
2   31,680
3   29,340
...
12  29,700
```

追加情報: 入金日は翌月25日、手数料なし、前年12月分入金あり（29,880円、1/29入金）

### 出力例

**売上CSV（売上_富田通商.csv）** — 0円の1月を除く11行:
```
"1","2025/02/20","売掛金","","","","","","31680","売上高","","","","","","31680","富田通商 2月分","",""
```

**入金CSV（入金_富田通商.csv）** — 前年12月分 + 2〜11月分の入金 = 11行:
```
"1","2025/01/29","普通預金","","","","","","29880","売掛金","","","","","","29880","富田通商 24/12月分入金","",""
"2","2025/03/25","普通預金","","","","","","31680","売掛金","","","","","","31680","富田通商 2月分入金","",""
```
※12月分の入金は翌年のため含めない
